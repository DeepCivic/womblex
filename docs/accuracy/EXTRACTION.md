## Metric Definitions

- **(raw)** = PaddleOCR on original image; **(pp)** = after preprocessing (deskew + binarise).
- **CER**: Levenshtein distance / reference length. 0.0 = perfect.
- **CER-s**: GT and OCR words spatially sorted by bounding-box centroid before CER. Isolates recognition from reading-order.
- **WER**: Word-level Levenshtein / reference word count.
- **Precision**: Predicted layout blocks matching a GT block with correct label (IoU >= 0.3).
- **Recall**: GT layout blocks matched by a prediction with correct label (IoU >= 0.3).
- **F1**: Harmonic mean of precision and recall.
- **Avg Confidence**: Mean per-region OCR confidence from RapidOCR (0�1).
